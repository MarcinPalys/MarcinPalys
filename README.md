## 🛠️ Featured Projects

### 🚗 Car Workshop  
🔗 [View on GitHub](https://github.com/MarcinPalys/CarWorkshop.git)  
A web application built using Clean Architecture and CQRS pattern.  
It handles commands and queries using MediatR, maps DTOs with AutoMapper, and validates data via FluentValidation.  
Database operations are managed by Entity Framework, and testing is done with xUnit.  
**Tech stack:** `C#` `ASP.NET MVC` `EF Core` `JavaScript` `MediatR` `CQRS` `xUnit`

### 📋 JobAggregator
🔗 [View on GitHub](https://github.com/MarcinPalys/ListaKontaktowWPF.git)  
A REST API that aggregates job offers from multiple sources.
Automatically fetches listings every hour and stores them in PostgreSQL with upsert deduplication. 
Built with Clean Architecture (Domain / Application / Infrastructure / API)..
**Tech stack:** `C#` `ASP.NET Core 9` `SQL` `ASP.NET Web API` `PostgreSQL` `Adzuna API`
